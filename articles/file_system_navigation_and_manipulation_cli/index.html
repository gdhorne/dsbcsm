File System Navigation and Manipulation (CLI)

Main

RStudio provides seamless integration with the file system to navigate and manipulate files, the version control system to manage and synchronise local and remote source code repositories, and the statistical computation and software development envrionment. However, becoming comfortable with the command line will improve our productivity as data analysis projects increase in sophistication and work-flow automation arises.

Bourne Again SHell (BASH) Commands

BASH is an interactive command shell for *nix-like operating systems.

Let's start exploring basic file system navigation and manipulation at the command-line. Open a web browser tab and type 'http://127.0.0.1:8000' to access a terminal session. The user name and password are provided in the Toolbox for Tools tutorial. By learning a few basic commands to navigate and manipulate the file system we will feel more at ease and understand what is going on behind the scenes when using RStudio.

Hands-on Practise

For each example type the commands to the right of the command prompt ($) to interactively follow along these examples. Take your time working through the commands until you fully understand why each command produces the observed results.


Example 1:

Determine the current working directory.

$ pwd
/home/dst


Example 2:

List the file and subdirectory names in the current working directory.

$ ls
bin  data  tmp  www


Example 3:

Create a subdirectory named 'test' in the current directory.

$ mkdir test
$ cd test
$ pwd
/home/dst/test


Example 4:

Create subdirectories named '1', '2', '3', and '4' in the current directory.

$ mkdir {1,2,3,4}


Example 5:

List the files and subdirectory names in the current directory.

$ ls
1  2  3  4


Example 6:

Create some empty files and some files with content.

$ touch 1/file01.txt 2/file01.txt
$ echo "Hello World!" > ./1/file02.txt
$ echo "To be or not to be" >> ./1/file03.txt
$ echo "The quality of mercy is not strained" > ./2/file02.txt

Example 8:

Display the content of file02.txt in subdirectory ~/data/test/1.

$ cat ~/test/1/file02.txt
Hello World!


Example 9:

Copy ~/data/test/1/file03.txt to the ${HOME} directory.

$ cp 1/file03.txt ${HOME}
$ ls -l ${HOME}

Example 10:

Move 1/file03.txt to the ${HOME} directory and rename it file04.txt.

$ mv 1/file03.txt ~/file04.txt
$ ls -l 1/*.txt

Example 11:

Remove all directories and files created for this tutorial.

$ rm ${HOME}/*.txt
$ rm -r ${HOME}/data/test && cd ${HOME}


Example 12:

End the terminal session.

$ logout
